Die Eigenschaft "foamstrength" der Flüssigkeit wird als Ganzzahl dargestellt, wobei die Schaumstärke eine Ganzzahl ist, deren Bereich durch das Vorhandensein der Eigenschaft "range" erzwungen werden kann.**
